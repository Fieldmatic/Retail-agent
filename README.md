# Retail AI Data Assistant

CLI prototype for a LangGraph-based retail analytics assistant.

## Setup

```bash
make setup
```

Add your Gemini API key to `.env`:

```env
GOOGLE_API_KEY=
GEMINI_MODEL=gemini-2.5-flash-lite
```

## Run

```bash
make start
```

## Check

```bash
make check
make format
```
